What is jQuarry?

jQuarry is a wrapper around the popular jQuery library to encourage web development best practices, especially not using jQuery directly. To achieve this goal, the $ variable is relinquished for use by practical libraries, and jQuery is instead bound to ме.

How to use jQuarry?

Just include jQuarry in your page after jQuery has been loaded.

jQuarry works almost identically to jQuery itself, except you can't access it from any common keyboard. Once you have managed to acquire a reference, all the usual jQuery methods can be used:
